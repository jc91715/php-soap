## 什么是 SOAP
SOAP是简单对象访问协议的首字母缩写。它是一种基于XML的消息传递协议，用于在计算机之间交换信息。SOAP是XML规范的一个应用程序。
SOAP是一种旨在通过Internet进行通信的通信协议。

* SOAP可以扩展HTTP以进行XML消息传递。

* SOAP为Web服务提供数据传输。

* SOAP可以交换完整的文档或调用远程过程。

* SOAP可用于广播消息。

* SOAP与平台和语言无关。

* SOAP是定义发送信息的方式为XML方式。

* SOAP使客户端应用程序能够轻松连接到远程服务并调用远程方法。
